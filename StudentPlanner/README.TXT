PROJECT TITLE: Student Planner
PURPOSE OF PROJECT: To create a Java GUI, connect to an SQL database and use various Object-Oriented programming such as inheritance to allow a student to view required courses for their program
HOW TO START THIS PROJECT: 
1. Git clone this repository to your computer.
2. Using blueJ open the folder with all the files up and right click on the Planner.java file.
3. Run the main.

OR after cloning to your computer run the Planner.java file like any other java file to begin the GUI.
